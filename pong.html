Pong

Left and Right arrow to move.

This game was created in my early scratch days, when I was creating a 2 player pong game. The ball physics is odd because the ball's bounce angle is random. With enough coding and some confusedness, I guess I could make it like the Atari Breakout (Click here to play) but I'm too lazy. Go figure.
Also, you can have some extra fun if you have a LEGO Mindstorms kit. First, connect it to scratch using this guide and plug in 2 touch sensors to 1 and 4 to play!
Also, you simply cannot score a point against the enemy. I have coded it that even if you force it to run across the map, it will do so. Speed is no constraint for the boss.
